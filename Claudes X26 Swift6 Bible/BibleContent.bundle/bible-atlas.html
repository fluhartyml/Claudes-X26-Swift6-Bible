  109. 12. How to Use This Atlas

  110. • To identify which file you’re looking at: the header and footer of every HTML doc shows its filename. Match it to a row in this atlas.

  111. • To point Claude at a file: “I’m on swift-section-mapping.html line 82” — or if the file name is long, “atlas line 17 — that one, line 82.”

  112. • To see relationships: the “Related” column of each row points at other entries. Each row is cross-referenced.

  113. • To claim / approve / edit an atlas row: “atlas line N yes” locks the entry; “atlas line N change to X” edits; “atlas line N huh?” prompts a plain-English explanation.

  114. • To add a new file to the vault: tell Claude, and the atlas gets a new row.
