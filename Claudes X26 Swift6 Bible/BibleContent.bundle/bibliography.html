  1. Bibliography

  2. Primary sources the book draws from. Individual claims are not yet footnoted line-by-line; a full citation audit is a later pass. Everything below is where to go for the authoritative word on the topics covered in each chapter.

  3. Last reviewed 2026-04-22.

  4. Swift Language

    • Apple Inc. The Swift Programming Language. docs.swift.org/swift-book. The canonical language reference; every keyword, type, and rule lives here.
    • Apple Inc. Swift.org. swift.org. Release notes, compiler announcements, and the Swift blog.
    • Swift Evolution. github.com/apple/swift-evolution. Every accepted language proposal, with motivation and alternatives considered.
    • Swift Forums. forums.swift.org. Where Swift evolution is debated in public.

  6. Apple Platform APIs

    • Apple Inc. Apple Developer Documentation. developer.apple.com/documentation. The authoritative source for every framework API surfaced in the book.
    • Apple Inc. SwiftUI. developer.apple.com/documentation/swiftui.
    • Apple Inc. SwiftData. developer.apple.com/documentation/swiftdata.
    • Apple Inc. Swift Charts. developer.apple.com/documentation/charts.
    • Apple Inc. PDFKit. developer.apple.com/documentation/pdfkit.
    • Apple Inc. PencilKit. developer.apple.com/documentation/pencilkit.
    • Apple Inc. UIKit. developer.apple.com/documentation/uikit.
    • Apple Inc. AppKit. developer.apple.com/documentation/appkit.
    • Apple Inc. Foundation. developer.apple.com/documentation/foundation.
    • Apple Inc. Core Transferable. developer.apple.com/documentation/coretransferable.

  8. Design and Platform Guidelines

    • Apple Inc. Human Interface Guidelines. developer.apple.com/design/human-interface-guidelines. Hit-target sizes, accessibility, platform-correct UI.
    • Apple Inc. App Store Review Guidelines. developer.apple.com/app-store/review/guidelines.

  10. WWDC Sessions

    • Apple Inc. WWDC Session Videos. developer.apple.com/videos. Primary source for each year's API additions. SwiftUI, SwiftData, Swift concurrency, and Swift Charts are particularly well served by the year-of-introduction sessions.

  12. Xcode and the Build System

    • Apple Inc. Xcode Release Notes. developer.apple.com/documentation/xcode-release-notes.
    • Apple Inc. Xcode Help. Bundled in-app via Xcode > Help > Xcode Help. Covers Signing & Capabilities, schemes, and build settings.

  14. Instruments and Performance

    • Apple Inc. Instruments User Guide. help.apple.com/instruments. Covers the Time Profiler, Allocations, and SwiftUI instruments cited in Book 20.

  16. Anthropic Claude API

    • Anthropic PBC. Anthropic API Reference. docs.anthropic.com. Messages API, streaming, rate limits, and model catalog used in Book 22.
    • Anthropic PBC. Anthropic Console. console.anthropic.com. API key management and usage dashboards.

  18. Git and GitHub

    • Chacon, Scott, and Straub, Ben. Pro Git, 2nd ed. Apress, 2014. git-scm.com/book. The standard free reference.
    • Git Project. git-scm.com. git-scm.com. Command reference and changelog.
    • GitHub Inc. GitHub Docs. docs.github.com. Remote-side workflows: pull requests, releases, actions.

  20. Other Languages Referenced (Rosetta Stone)

    • ISO/IEC 14882:2020. Programming languages — C++. International Organization for Standardization, 2020. Used for C/C++ comparisons in the Lexicon.
    • ISO/IEC 7185:1990. Information technology — Programming languages — Pascal. Cited for the Pascal/Delphi cousins in the Lexicon.
    • Embarcadero Technologies. Delphi Language Reference. docwiki.embarcadero.com/RADStudio.
    • Microsoft Corporation. QuickBASIC Language Reference. Historical reference for the BASIC dialect used in the Lexicon comparisons.

  22. Fonts and Assets

    • Van Rossum, Nikita. Fira Code. github.com/tonsky/FiraCode. Monospaced font used throughout the book and the reader app.
    • Ryan L. McIntyre and Fira Code contributors. Nerd Fonts. nerdfonts.com. Patched font glyphs.

  24. Notes on Currency and Accuracy

  25. This edition is written for Xcode 26 / Swift 6. Anything API-shaped moves between Apple releases; when a call site in the book disagrees with the linked primary source, the primary source is authoritative. The next major edition will be cut after WWDC 26 and updated to Xcode 27 / Swift 7.
