  1. infix

  2. Kind: keyword

  3. Definition

  4. Declares that a custom operator sits between its two operands (like `+`, `==`, `..<`). Without `infix`, the compiler doesn't know how to parse your operator.

  5. Swift Example

  6. infix operator **: MultiplicationPrecedence

func ** (base: Int, exp: Int) -> Int {
    var r = 1
    for _ in 0..<exp { r *= base }
    return r
}

print(2 ** 10) // 1024
