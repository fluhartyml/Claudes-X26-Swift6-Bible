Part II The Swift Language/Chapter D/Page Decodable

  1. Decodable

  2. Kind: protocol

  3. Definition

  4. A protocol that describes a type that can be created from an external representation (like JSON). A decoder (e.g. `JSONDecoder`) reads the representation and produces an instance. Conformance is synthesized for types whose stored properties are all `Decodable`.

  5. Swift Example

  6. struct Point: Decodable { let x: Double; let y: Double }

let json = #"{ "x": 1.5, "y": 2.5 }"#.data(using: .utf8)!
let p = try JSONDecoder().decode(Point.self, from: json)
print(p.x, p.y) // 1.5 2.5

  7. Rosetta Stone

  8. Pascal / Delphi

  9. No cousin. Manual deserialization via `TJsonReader` / `TJSONObject.ParseJSONValue`.

  10. BASIC

  11. `<Serializable>` plus `DataContractSerializer.ReadObject` / `JsonSerializer.Deserialize`.

  12. C / C++

  13. No cousin. Parsing libraries (cJSON, nlohmann::json) with explicit per-field extraction.
