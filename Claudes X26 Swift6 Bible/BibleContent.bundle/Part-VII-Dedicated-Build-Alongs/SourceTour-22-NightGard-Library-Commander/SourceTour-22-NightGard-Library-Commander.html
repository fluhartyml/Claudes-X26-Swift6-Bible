Source Tour 22: NightGard Library Commander

Status: NOT YET LIVE — must reach the Mac App Store before book official release. Mac app. App-Store-compliant (sandboxed). Companion to CryoTunes Player.

Source tour for NightGard Library Commander. Paired with CryoTunes Player — two-app ecosystem showing how a Mac-side library tool works alongside an iOS-side player through the shared iCloud Music Library.

What Library Commander shows at production scale

  • Apple Music library metadata cleanup (fills missing artist/album/genre, handles problem-state tracks)
  • MusicKit + AppleScript via the sandbox-with-temporary-exception pattern
  • Security-scoped bookmarks for user-picked working folders
  • SwiftData for review queue state, CloudKit sync optional

Ecosystem pairing with CryoTunes Player

Library Commander cleans the library on Mac. CryoTunes plays from that cleaner library on iPhone. Both share the same iCloud Music Library. Readers see how production Mac + iOS apps compose into an ecosystem, not just standalone experiences.
